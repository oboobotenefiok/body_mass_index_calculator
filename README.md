# Body Mass Index Calculator

###### Hi friend, this is my second (self-started) project in Rust on here.

It's a simple command-line Body Mass Index (BMI) calculator written in Rust, originally intended for medical practitioners but can be used by anyone.

## Description

This program calculates your Body Mass Index (BMI) based on your height (in meters) and weight (in kilograms). It's a straightforward implementation that demonstrates basic Rust concepts including:

- · · Structs and implementations
- · · User input handling
- · · Error handling with pattern matching
- · · Basic arithmetic operations

## Formula

The BMI is calculated using the standard formula:

```
BMI = weight (kg) / (height (m) × height (m))
```

## Features Include

- · · Interactive command-line interface
- · · Input validation for numeric values
- · · Clear error messages for invalid input
- · · Formatted output with 2 decimal places

## Installation

1. Ensure you have Rust installed on your system with Cargo
2. Clone this repository:
   ```bash
   git clone https://github.com/oboobotenefiok/body_mass_index_calculator.git
   ```
3. Navigate to the project directory:
   ```bash
   cd body_mass_index_calculator
   ```

## Usage

Run the program using Cargo:

```bash
cargo run
```

The program will prompt you for:

1. Your height in meters (e.g., 1.75)
2. Your weight in kilograms (e.g., 70.5)

It will then calculate and display your BMI.

## Example Round

```bash
$ cargo run
What's your height in metres?
1.75
You entered 1.75 

Now, what's your weight in Kilograms?
70.5
You entered 70.5

Your Body Mass Index is: 23.02
```

## BMI Categories (Reference)

According to the World Health Organization:

Category BMI Range (kg/m²)

- · Underweight < 18.5
- · Normal weight 18.5 – 24.9
- · Overweight 25 – 29.9
- · Obesity ≥ 30

## Disclaimer 
This is a simple calculator and does not provide medical advice. BMI has limitations and doesn't account for muscle mass, bone density, overall body composition, or racial and sex differences.

## Code Overview

The main components are:

1. Body struct: Contains mass and height fields
2. Body::bmi() method: Calculates BMI using the struct's data
3. main() function: Handles user input, creates a Body instance, and displays the result
